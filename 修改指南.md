## 一、页面整体结构

```

├── 左侧悬浮导航条（6个圆点，滚动跟踪+点击跳转）
├── 左侧固定侧边栏（头像、名字、副标题、链接）
│   ├── avatar → KX
│   ├── h1 → K.X.
│   ├── subtitle → Electrical Engineering & AI
│   └── sidebar-links → GitHub / Email
└── 右侧主内容区
    ├── About → 自我介绍
    ├── Education → 学历
    ├── Experience → 工作/项目经历
    ├── Research Interests → 研究兴趣标签
    ├── Technical Skills → 技能标签
    ├── Awards & Honors → 获奖荣誉
    ├── Contact → 联系方式按钮
    └── Footer → 底部版权
```

## 二、三种修改方式

### 方式1：GitHub网页直接改（最简单）

1. 打开 https://github.com/TpoarSerx/TpoarSerx.github.io
2. 点击要改的文件（如 `index.html`）
3. 点右上角 ✏️ 铅笔图标
4. 改完后点右上角绿色 **Commit changes**
5. 等1-2分钟刷新线上页面生效

### 方式2：本地改后推送

```bash
# 首次克隆（只需一次）
git clone https://github.com/TpoarSerx/TpoarSerx.github.io.git
cd TpoarSerx.github.io

# 用任意编辑器修改文件（VS Code推荐）

# 推送到线上
git add .
git commit -m "描述你改了什么"
git push
```

### 方式3：告诉KK改

直接说就行，比如"About里加一行：竹笛爱好者"，我帮你改完直接推上去。

## 三、常见修改操作

### 3.1 改名字/副标题/头像字母

找到 `index.html` 中侧边栏部分：

```html
<!-- 改头像里的字母 -->
<div class="avatar">KX</div>          ← 改成你想要的，如 XQ、相框 等

<!-- 改名字 -->
<h1>K.X.</h1>                         ← 改成你的署名

<!-- 改副标题 -->
<p class="subtitle">Electrical Engineering & AI<br>Energy Systems</p>
                                       ← 改成你想显示的标题
```

### 3.2 改邮箱/GitHub链接

```html
<!-- 改邮箱 -->
<a href="mailto:contact@example.com">  ← 把 contact@example.com 换成你的邮箱

<!-- 改GitHub -->
<a href="https://github.com/TpoarSerx" target="_blank">
                                       ← 换成你的GitHub主页地址
```

**侧边栏和Contact区都有邮箱，两处都要改。**

### 3.3 修改About自我介绍

找到 `<section id="about">` 部分：

```html
<section id="about">
    <h2>About</h2>
    <p>
        Electrical engineering student with a passion for the intersection of 
        <strong>energy systems</strong> and <strong>artificial intelligence</strong>. 
        ...
    </p>
    <p>
        I enjoy understanding how cities evolve ...
    </p>
</section>
```

- 每段文字用 `<p>...</p>` 包裹
- 加粗用 `<strong>文字</strong>`
- 换段就再加一个 `<p>...</p>`

### 3.4 修改Education学历

```html
<section id="education">
    <h2>Education</h2>
    <div class="entry">
        <div class="entry-header">
            <span class="title">Hubei Polytechnic University</span>    ← 学校名
            <span class="date">2026.09 – 2028.06</span>               ← 时间
        </div>
        <div class="detail">B.Eng. in Electrical Engineering & Automation</div>  ← 学位专业
    </div>
    <!-- 再加一条学历：复制上面的 <div class="entry">...</div> 即可 -->
</section>
```

### 3.5 修改Experience经历

```html
<section id="experience">
    <h2>Experience</h2>
    <div class="entry">
        <div class="entry-header">
            <span class="title">Energy Management System Operator</span>  ← 职位/角色
            <span class="date">2025 – Present</span>                      ← 时间
        </div>
        <div class="detail">
            Hong'an Cigarette Factory · Operations & Maintenance<br>      ← 单位<br>换行
            <small>Industrial IoT data monitoring · ...</small>           ← 补充说明
        </div>
    </div>
</section>
```

- `<br>` 是换行
- `<small>...</small>` 是灰色小字说明
- 加多条经历就复制 `<div class="entry">...</div>`

### 3.6 修改标签（Research Interests / Skills）

```html
<div class="tags">
    <span class="tag">Smart Grid</span>          ← 一个标签
    <span class="tag">Energy Management</span>    ← 再加就复制这行
</div>
```

直接增删 `<span class="tag">标签文字</span>` 即可。

### 3.7 修改Awards获奖

```html
<ul class="awards">
    <li>🏆 National Competition Award (College Level)</li>    ← 一条
    <li>🎓 Merit-based Admission to Bachelor's Program</li>  ← 再加就复制
</ul>
```

### 3.8 添加新板块

在 `</section>` 和下一个 `<section>` 之间插入：

```html
<!-- 例：添加 Projects 板块 -->
<section id="projects">
    <h2>Projects</h2>
    <div class="entry">
        <div class="entry-header">
            <span class="title">项目名称</span>
            <span class="date">2026</span>
        </div>
        <div class="detail">项目描述文字</div>
    </div>
</section>
```

**如果新板块也想出现在侧边导航条里**，还需要两步：

1. 在 `<nav class="side-nav">` 里加一个导航点：
```html
<a href="#projects" class="nav-dot" data-section="projects" title="Projects">
    <span class="dot"></span>
    <span class="nav-label">Projects</span>
</a>
```

2. 确保 section 有对应的 id：`<section id="projects">`

### 3.9 删除某个板块

直接删掉对应的整个 `<section>...</section>` 即可。如果该板块在导航条里有对应圆点，也一起删掉。

## 四、样式调整（style.css）

### 4.1 改主题色

在 `style.css` 中搜索 `#667eea`（紫色），全部替换为你想要的颜色：

| 颜色 | 色值 |
|------|------|
| 蓝色 | `#2563eb` |
| 绿色 | `#059669` |
| 红色 | `#dc2626` |
| 橙色 | `#ea580c` |
| 青色 | `#0891b2` |

涉及位置：
- 导航圆点高亮：`.nav-dot.active .dot`
- 标签hover：`.tag:hover`
- 侧边链接hover：`.sidebar-links a:hover`
- 头像渐变：`.avatar` 的 `linear-gradient`

### 4.2 改头像渐变

```css
.avatar {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    /*                    ↑ 起始色         ↑ 结束色           */
}
```

### 4.3 改字体大小

```css
.sidebar h1 { font-size: 1.5rem; }     /* 侧边栏名字 */
h2 { font-size: 1.1rem; }              /* 板块标题 */
p { font-size: 0.92rem; }              /* 正文 */
```

### 4.4 改导航条位置

```css
.side-nav {
    left: 24px;        ← 距左边距离，改大就往右挪
    top: 50%;          ← 垂直居中
    gap: 20px;         ← 圆点间距
}
```

## 五、进阶操作

### 5.1 添加真实头像图片

1. 把图片文件（如 `avatar.jpg`）放到仓库根目录
2. 修改 `index.html`：

```html
<!-- 替换原来的 <div class="avatar">KX</div> -->
<img class="avatar" src="avatar.jpg" alt="K.X.">
```

3. 修改 `style.css`：

```css
.avatar {
    width: 100px;
    height: 100px;
    border-radius: 50%;
    object-fit: cover;
    margin: 0 auto 16px;
    box-shadow: 0 4px 15px rgba(102, 126, 234, 0.25);
}
```

### 5.2 添加论文/项目链接

```html
<section id="publications">
    <h2>Publications</h2>
    <ul class="awards">
        <li>📄 <a href="论文链接" target="_blank">论文标题</a> — 期刊/会议, 2026</li>
    </ul>
</section>
```

### 5.4 绑定自定义域名

1. 在域名服务商把域名 CNAME 指向 `tpoarserx.github.io`
